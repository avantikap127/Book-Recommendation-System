# BookRecommendation

Book Recommendation System - demonstrates Spring Boot, JPA, JDBC, REST, file export, exceptions.

Run:
```
mvn spring-boot:run
```

API examples:
- POST /api/books
- GET /api/books
- GET /api/books/recommend
- GET /api/books/export  -> downloads reading-list.txt

H2 console: /h2-console (jdbc:h2:mem:testdb)
